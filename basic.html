first session of the training we covered about git and github. We learned about the GUI and The CLI, we covered

  1. Initialize a Git repository: `git init`
  2. Check the status of the repository: `git status`
  3. Add files to the staging area: `git add `
  4. Commit changes: `git commit -m "message"`
  5. Check commit history: `git log`
  6. Show differences between commits: `git diff `
  7. Push changes to a remote repository: `git push `
  8. Pull changes from a remote repository: `git pull `
  9. Create a branch: `git branch `
  10. Switch to a branch: `git checkout `
  11. Merge a branch into the current branch: `git merge `
  12. Reset changes to a specific commit: `git reset --hard `
